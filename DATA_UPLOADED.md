##  Команды для работы с данными

### Переименовать файлы
```bash
python3 rename_images.py data/крокодил крокодил
python3 rename_images.py data/аллигатор аллигатор
python3 rename_images.py data/кайман кайман
```

### Загрузить в S3
```bash
# Один класс
python3 s3_upload.py --dir data/крокодил media/images/крокодил/

# Все классы
python3 s3_upload.py --dir data/аллигатор media/images/аллигатор/
python3 s3_upload.py --dir data/кайман media/images/кайман/
```

### Посмотреть файлы в S3
```bash
# Все файлы
python3 s3_upload.py --list ""

# Только изображения
python3 s3_upload.py --list "media/images/"

# Конкретный класс
python3 s3_upload.py --list "media/images/крокодил/"
```
