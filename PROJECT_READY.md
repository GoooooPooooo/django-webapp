##  Быстрые команды

### Запуск приложения
```bash
# Если сервер ещё не запущен
cd homework
python3 web-site-dl/manage.py runserver 0.0.0.0:8000
```

### Управление через Makefile
```bash
make help              # Показать все команды
make docker-up         # Запустить Docker с MinIO
make docker-down       # Остановить Docker
make upload-all        # Загрузить данные в S3
make s3-list           # Показать файлы в S3
```

##  Что дальше?

### 1⃣ Тестирование веб-приложения
1. Откройте http://localhost:8000
2. Загрузите любое изображение
3. Нажмите "Классифицировать"
4. Получите результат

**Примечание:** Сейчас модель не подключена (нужно обучить в Colab).

### 2⃣ Обучение модели (Google Colab)
1. Откройте `colab_notebook.ipynb` в Google Colab
2. Загрузите реальные изображения (≥100 на класс)
3. Запустите все ячейки
4. Скачайте `cifar100.onnx`
5. Положите модель в `web-site-dl/media/models/`

### 3⃣ Скачивание реальных изображений
См. инструкцию: **[DATA_DOWNLOAD.md](DATA_DOWNLOAD.md)**

Кратко:
1. Установите расширение Image downloader - Imageye для Chrome
2. Скачайте изображения: крокодил, аллигатор, кайман
3. Разложите в папки: `data/крокодил/`, `data/аллигатор/`, `data/кайман/`
4. Загрузите в S3: `make upload-all`
