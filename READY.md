## 📁 Распаковка архивов

### Команда для распаковки и переименования:

```bash
# Распаковать ZIP архив (игнорируя пути)
unzip -o -j крокодил.zip -d data/крокодил/

# Переименовать файлы
python3 extract_and_rename.py UNUSED data/крокодил/ крокодил

# Загрузить в S3
python3 s3_upload.py --dir data/крокодил media/images/крокодил/
```

### Через Makefile:

```bash
# Распаковать и переименовать
make extract-archive ARCHIVE=data/крокодил.zip FOLDER=data/крокодил/ PREFIX=крокодил

# Загрузить в S3
python3 s3_upload.py --dir data/крокодил media/images/крокодил/
```
