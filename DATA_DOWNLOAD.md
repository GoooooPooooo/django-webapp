#  Инструкция по скачиванию изображений

## Проблема
Яндекс.Картинки и Google Images блокируют автоматическое скачивание изображений.

##  Решение 1: Скачать вручную через браузер (РЕКОМЕНДУЕТСЯ)

### Шаг 1: Установите расширение для Chrome

**Image downloader - Imageye**
- Ссылка: https://chrome.google.com/webstore/detail/image-downloader-imageye/agionbommeaifngbhincahgmoflcikhm
- Или: https://imageye.net/

**Альтернатива: Image Downloader**
- Ссылка: https://chrome.google.com/webstore/detail/image-downloader/cnpniohnfphhjihaiiggeabnkjhpaldj

### Шаг 2: Найдите изображения

1. Откройте Яндекс.Картинки или Google Images
2. Введите поисковый запрос:
   - `крокодил` (или `crocodile`)
   - `аллигатор` (или `alligator`)
   - `кайман` (или `caiman`)

### Шаг 3: Скачайте изображения

1. Откройте расширение Imageye
2. Выберите фильтр **JPG** (рекомендуется)
3. Нажмите **Select All** или выберите нужные изображения
4. Нажмите **Download**
5. Изображения скачаются в папку `Downloads`

### Шаг 4: Организуйте файлы

Создайте структуру папок:
```
homework/data/
├── крокодил/
│   ├── крокодил_001.jpg
│   ├── крокодил_002.jpg
│   └── ...
├── аллигатор/
│   ├── аллигатор_001.jpg
│   ├── аллигатор_002.jpg
│   └── ...
└── кайман/
    ├── кайман_001.jpg
    ├── кайман_002.jpg
    └── ...
```

**Минимум 100 изображений на каждый класс!**

##  Решение 2: Использовать готовые датасеты

### Kaggle Datasets
1. Перейдите на https://www.kaggle.com/datasets
2. Найдите по запросам:
   - `crocodile dataset`
   - `alligator dataset`
   - `reptile classification`
3. Скачайте датасет
4. Разложите изображения по папкам согласно классам

### iNaturalist Dataset
- https://www.inaturalist.org/observations
- Бесплатный датасет с изображениями животных

##  Решение 4: Установить Chrome для Selenium

Если хотите использовать автоматическое скачивание:

```bash
# Установка Chromium (Ubuntu/Debian)
sudo apt update
sudo apt install -y chromium-browser chromium-chromedriver

# Проверка
chromium-browser --version
```

Затем запустите:
```bash
make download-all
```

##  Требования к изображениям

| Параметр | Требование |
|----------|------------|
| Количество | ≥100 на класс |
| Формат | JPG, PNG |
| Размер | Любой (будет масштабировано до 32x32) |
| Качество | Чёткое, объект виден хорошо |

---

##  Важные замечания

1. **Удалите дубликаты** - проверьте папки и удалите повторяющиеся изображения
2. **Проверьте тематику** - убедитесь, что изображения соответствуют классу
3. **Разные ракурсы** - желательно иметь изображения с разных углов
4. **Разное освещение** - для лучшей генерализации модели

После скачивания проверьте:

```bash
# Посчитать количество изображений
ls data/крокодил/*.jpg | wc -l
ls data/аллигатор/*.jpg | wc -l
ls data/кайман/*.jpg | wc -l

# Просмотреть первые файлы
ls data/крокодил/ | head
```
